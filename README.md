# KitchenOS

Sistema web desenvolvido para auxiliar no gerenciamento operacional de restaurantes, centralizando informações relacionadas a pedidos, mesas e atividades internas do estabelecimento.

## Visão Geral

O KitchenOS foi criado com o objetivo de otimizar processos internos de restaurantes por meio de uma interface intuitiva e responsiva. A aplicação busca reduzir falhas operacionais, melhorar a organização dos pedidos e facilitar o acompanhamento das atividades realizadas pelos colaboradores.

## Problema Resolvido

Muitos restaurantes ainda realizam o controle de pedidos e mesas de forma manual ou utilizando ferramentas desconectadas, o que pode gerar:

* Falhas na comunicação entre equipes;
* Atrasos no atendimento;
* Dificuldade no acompanhamento dos pedidos;
* Problemas na organização das operações internas.

O KitchenOS centraliza essas informações em um único ambiente digital, tornando o gerenciamento mais eficiente.

## Objetivos

### Objetivo Geral

Desenvolver um sistema web capaz de auxiliar no gerenciamento operacional de restaurantes.

### Objetivos Específicos

* Gerenciar pedidos de forma organizada;
* Controlar mesas do estabelecimento;
* Acompanhar o status dos pedidos;
* Oferecer uma interface simples e intuitiva;
* Aplicar conceitos de desenvolvimento web moderno.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript ES6+
* Bootstrap 5
* jQuery
* Vite

## Funcionalidades

* Gerenciamento de pedidos;
* Controle de mesas;
* Acompanhamento visual dos pedidos;
* Interface responsiva;
* Navegação intuitiva;
* Estrutura modular para futuras expansões.

## Arquitetura do Projeto

A aplicação foi desenvolvida utilizando uma arquitetura front-end baseada em componentes, com separação entre estrutura, estilização e comportamento.

* HTML: Estrutura das páginas;
* CSS e Bootstrap: Interface e responsividade;
* JavaScript e jQuery: Regras de negócio e interações;
* Vite: Ambiente de desenvolvimento e build.

## Resultados Obtidos

O projeto alcançou os objetivos propostos, entregando uma aplicação funcional para apoio ao gerenciamento operacional de restaurantes.

Entre os principais resultados estão:

* Melhor organização dos pedidos;
* Centralização das informações;
* Interface adaptável a diferentes dispositivos;
* Aplicação prática dos conhecimentos adquiridos durante a disciplina.

## Desafios Encontrados

Durante o desenvolvimento foram enfrentados desafios relacionados a:
